<h3>Kurulum</h3>
 
1. Projeyi Klonlayın
```sh
git clone https://github.com/muhammeddeniz/working-area.git
```
2. Yarn Paketlerini İndirin
```sh
yarn
```

## Geliştirme 

```sh
yarn
yarn dev
```

http://localhost:3000/ adresinden projeye ulaşabilirsiniz
